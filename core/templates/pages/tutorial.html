Com funciona el formulari?

  1. 1. Introdueix les dades generals del projecte

    El formulari comença amb una pantalla on cal omplir les dades bàsiques (nom del projecte, fase, localització, etc.). Tots els camps són obligatoris, i ajuden a generar un informe més complet i contextualitzat.

  2. 2. Navega per les diferents dimensions

    Un cop completada la primera pantalla, podràs accedir a les seccions temàtiques de cada dimensió utilitzant els botons “Següent” i “Anterior”, que trobaràs a la part inferior de cada pàgina.

  3. 3. Decideix quines seccions vols valorar

    No cal completar totes les seccions. Pots saltar aquelles que no t’interessin.
    Però atenció: si vols obtenir una puntuació per una secció concreta, has de respondre totes les preguntes obligatòries d’aquella secció.

  4. 4. Tingues en compte els missatges d’instrucció

    A cada secció poden aparèixer indicacions específiques. Si una secció no apareix als resultats finals, probablement és perquè no s’han omplert tots els camps obligatoris.

  5. 5. Finalitza el formulari

    Quan arribis a l’última secció de la dimensió ambiental, veuràs el botó “Finalitzar”. En fer clic, es calcularà l’índex i accediràs automàticament a la vista de resultats.

  6. 6. Consulta els resultats

    A la vista de resultats trobaràs:
    • La puntuació global i normalitzada del projecte
    • Les dades generals introduïdes
    • Un gràfic per a cada dimensió amb la seva puntuació
    • I un botó desplegable per consultar els detalls de cada secció valorada
